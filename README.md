## Welcome to Mapping_Earthquakes

1. Retrieve data from a GeoJSON file.
2. Populate geographical maps with GeoJSON data using JavaScript and the Data-Driven Documents (D3) library.
3. Add multiple map layers to geographical maps using Leaflet control plugins to add user interface controls.
4. Use JavaScript ES6 functions to add GeoJSON data, features, and interactivity to maps.
5. Render maps on a local server.
